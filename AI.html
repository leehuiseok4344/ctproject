Project Summary

Expected Input

신용장 개설은행이 발행한 신용장 원문

Expected Output

대금회수가 가능한지 여부 (YES/NO), 문제 조건 제시

Topics and Core Ideas

AI를 활용하여 무역 거래에서 사용되는 신용장(Letter of Credit, L/C)의 조건을 자동으로 분석하고, 수출업자의 관점에서 대금 회수가 가능한지 여부를 판단한다. 이 문제는 국제상무 수업에서 실제로 다루는 '신용장의 조건 확인' 단원을 바탕으로 하며, 조건의 적합성 여부를 판단하는 데 컴퓨팅 사고 기반 문제 해결력이 필요하다.

Data Collection

• 샘플 L/C 조건:

• 선적일: 2025.08.30

• 서류제시기한: 선적일 후 21일

• 신용장 만기일: 2025.09.25

• 제시서류: 상업송장, 선하증권, 원산지증명서 등

List of Sample Data

Important Notice:

  • 1. If using real data, personally identifiable information has been removed
  • 2. Data used solely for educational purposes in this course
  • 3. Open data sources preferred (data.gov, Kaggle, etc.)

• ICC 샘플 L/C
